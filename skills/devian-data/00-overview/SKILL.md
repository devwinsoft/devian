# devian-data — Overview

Devian 테이블, Contract, 스토리지 포맷을 담당한다.

- **Table Authoring**: Excel 테이블 작성 규칙
- **Contract**: JSON Row I/O, Class Cell 포맷
- **Storage**: NDJSON, PB64 바이너리 포맷
- **Codegen**: TableGen, ContractGen, EnumGen 구현
