# devian-data Policy

## Scope
This folder contains DATA-related documents:
- Table authoring rules and cell formats (DFF)
- Row I/O and storage formats (NDJSON, pb64)
- String table conventions
- TableGen, ContractGen, and EnumGen implementation

These documents focus on **specification (what)**, not build pipeline mechanics.
For build/workspace tooling, see `skills/devian-tools/`.

## SSOT Reference
The Single Source of Truth (SSOT) is at `skills/devian-core/03-ssot/SKILL.md`.
