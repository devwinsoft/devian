# Devian Common Skills

Devian.Common 모듈 및 Common 도메인에 대한 스킬 문서.

## 스킬 목록

| 스킬 | 설명 |
|------|------|
| `00-skill-specification` | Common 스킬 전반 규약 |
| `12-common-standard-types` | 공용 표준 타입 (`G*` prefix) |
| `70-common-domain-policy` | Common 도메인 기본 정책 |

## 주요 경로

### C#

```
framework/cs/Devian.Common/
├── generated/   # 기계 생성
└── manual/      # 사람 작성
```

### TypeScript

```
framework/ts/devian-common/
├── generated/   # 기계 생성
└── manual/      # 사람 작성
```

## 참조

- Common 도메인: `input/Common/`
- 상위 스킬: `skills/devian/`
