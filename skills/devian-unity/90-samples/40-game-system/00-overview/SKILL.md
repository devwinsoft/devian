# devian-unity/40-game-system — Overview

Status: ACTIVE
AppliesTo: v10

Game System은 **Devian Samples**(`com.devian.samples`)에 포함된 Unity Sample이다.
이 그룹은 Game 도메인/컨텐츠 샘플(및 그 하위 샘플들)의 스킬을 담당한다.
