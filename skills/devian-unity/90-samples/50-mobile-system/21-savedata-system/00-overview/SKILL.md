# 21-savedata-system — Overview


Status: ACTIVE
AppliesTo: v10


SaveData System은 로컬/클라우드 저장 및 동기화를 위한 스킬 그룹이다.
SaveDataManager가 유일한 진입점이며, SaveLocal/SaveCloud 로직을 단일 매니저로 통합한다.
