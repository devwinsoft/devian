# 14-effect-system — Overview

Status: ACTIVE
AppliesTo: Devian Unity

## Scope

이 그룹은 Devian Unity의 Effect 시스템(공통 이펙트/머티리얼 이펙트)을 다룬다.

- CommonEffect: 공통 이펙트 스폰/디스폰 + 플레이어 확장
- MaterialEffect: Renderer Material[] 스위치 + MaterialEffectAsset 참조 ID
