# 22-sound-system — Overview

Status: ACTIVE
AppliesTo: Devian Unity

## Scope

이 그룹은 Devian Unity의 사운드 시스템(테이블 규약/재생 매니저/보이스 resolve/도메인/오디오 베이스)을 다룬다.

- Tables: Sound/Voice 테이블 규약
- Runtime: SoundManager, BaseAudioManager
- Resolve: 언어/리소스 경로 결정(Voice Table Resolve)
- Domain: Sound Domain 모델/정책
