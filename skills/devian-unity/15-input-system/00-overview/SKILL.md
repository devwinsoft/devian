# 15-input-system — Overview

Status: ACTIVE
AppliesTo: Devian Unity

## Scope

이 그룹은 Devian Unity의 입력 시스템(Input)을 다룬다.

- InputManager: 입력 상태 집계/라우팅
- InputController: 입력 제어/구성(플랫폼/디바이스 등)
- VirtualGamepad: 가상 게임패드(모바일 UI 입력)
