# devian-save — Overview


Devian의 **Cloud Save + Local Save (Unity 전용)** 스킬 그룹.


목표는 "직접 서버 운영 없이" **Unity**에서 다음 플랫폼 네이티브 Cloud Save와
동일 스키마의 Local Save를 사용해 미니게임을 출시 가능하게 만드는 것:


- Android: Google Play Games Saved Games
- iOS: iCloud
- PC: Steam Cloud (Remote Storage)


비목표:
- 서버/BaaS 없이 **크로스플랫폼(안드↔iOS↔Steam) 세이브 공유**를 보장하는 것
