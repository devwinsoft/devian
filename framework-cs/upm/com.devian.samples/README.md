# Devian Samples

**Package:** `com.devian.samples`

## 개요

Devian 프레임워크용 샘플 모음. Unity Package Manager에서 원하는 샘플을 설치하고 자유롭게 수정하세요.

## 설치 방법

1. Unity 프로젝트 열기
2. Window → Package Manager
3. `Devian Samples` 패키지 선택
4. Samples 섹션에서 원하는 샘플 "Import" 클릭

## 포함된 샘플

### Network
WebSocket 클라이언트 샘플. Protocol 연동 예시 포함.

- **assembly (asmdef):** Devian Templates Network
- **namespace:** `Devian`
- **의존성:** com.devian.unity, com.devian.protocol.sample

## 설치 후

샘플은 Unity가 생성하는 복사본 폴더(`Assets/Samples/...`)에 저장됩니다.
이 코드는 프로젝트 소유이므로 자유롭게 수정할 수 있습니다.
